Repozytoria Git

System kotnroli werjsi - jest to stystem pozwalający na sprawdzanie zmianw kodzie źródłowym oraz pomaga programistom w pracy zespołowej.
Aby zacząć korzystać z GIT-a wystarczy pobrać Git brush. Można utworzyć projekt komendą lub otwrzyć folder za pomogcą Git Brush. Wtedy wystarczy w konsoli wpisać '$git init' i repozytorium zostało utworzone.
